Eclipse Import Instructions
===========================
This project includes Eclipse metadata files:
- .project
- .classpath
- .settings/org.eclipse.jdt.core.prefs

Recommended steps:
1. Extract this ZIP.
2. In Eclipse: File -> Import -> Existing Projects into Workspace
3. Select the extracted project folder.
4. Finish.
5. Run src/Main.java as a Java Application.

Java version:
- Configured for JavaSE-17

Notes:
- Source folder: src
- Output folder: bin
- Data files live in the project root under data/
